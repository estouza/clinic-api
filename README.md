<h1>EN</h1>

# A Brighter Day Clinic
Smart Appointment Management System for Medical Clinics

This project consists of the development of a web application for managing medical appointments, designed to support small clinics that need a simple, secure, and efficient solution to organize their scheduling processes.

The application allows secure user registration and authentication, appointment scheduling with availability verification, and integration with external APIs to enhance the user experience. Among the integrations are automatic address lookup through a ZIP code API and weather forecast verification for the day of the appointment, allowing the system to alert users about possible rain.

The system was developed using Node.js and Express for the backend and Vue.js for the frontend, following good coding practices, implementing JWT authentication, and integrating with a database to store user and appointment information.

Features

* Secure user registration and login
* Authentication using JSON Web Token (JWT)
* Appointment scheduling with time slot control
* Automatic address lookup using a ZIP code API
* Integration with a weather forecast API
* Administrative dashboard for managing appointments

<h2>Technologies Used</h2>

* Node.js
* Express
* MongoDB
* JWT (JSON Web Token)
* Vue.js
* External APIs (ZIP code and weather)

<h2>Objective</h2>

The objective of this project is to apply concepts of web development, authentication, API integration, and application architecture, resulting in a functional system ready for deployment and practical use.

<h1>PT</h1>
# Clínica Pro Dia Nascer Feliz
Sistema de Atendimento Inteligente para Clínicas Médicas

Este projeto consiste no desenvolvimento de uma aplicação web para gerenciamento de consultas médicas, criada para atender clínicas de pequeno porte que buscam uma solução simples, segura e eficiente para organizar seus atendimentos.

A aplicação permite o cadastro e autenticação segura de usuários, agendamento de consultas com verificação de disponibilidade, além da integração com APIs externas para enriquecer a experiência do usuário. Entre as integrações estão a consulta automática de endereço via API de CEP e a verificação da previsão do tempo no dia da consulta, permitindo alertas sobre possibilidade de chuva.

O sistema foi desenvolvido utilizando Node.js e Express no backend e Vue.js no frontend, seguindo boas práticas de organização de código, autenticação com JWT, e integração com banco de dados para armazenamento das informações de usuários e consultas. 


<h2>Funcionalidades</h2>

* Cadastro e login seguro de usuários
* Autenticação com JSON Web Token (JWT)
* Agendamento de consultas com controle de horários
* Consulta automática de endereço via CEP
* Integração com API de previsão do tempo
* Painel administrativo para gerenciamento de atendimentos

<h2>Tecnologias utilizadas</h2>

* Node.js
* Express
* MongoDB
* JWT (JSON Web Token)
* Vue.js
* APIs externas (CEP e clima)

<h2>Objetivo</h2>

O objetivo do projeto é aplicar conceitos de desenvolvimento web, autenticação, integração com APIs e arquitetura de aplicações, resultando em um sistema funcional pronto para deploy e utilização.
